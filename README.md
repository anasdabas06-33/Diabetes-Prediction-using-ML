# Diabetes Prediction using ML 
This project predicts whether a patient is likely to have diabetes using machine learning algorithms.

## Usage
1. Input patient data (e.g., glucose, BMI, age).
2. The model outputs whether the patient is likely diabetic.

## Features
- Data preprocessing
- Model training
- Performance evaluation
- Prediction system

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
